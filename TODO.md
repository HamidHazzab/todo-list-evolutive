# To-Do List Project - Task Planning

## MVP Features
- Add a new task
- Delete a task
- Mark task as completed / not completed
- Display tasks in a list

## Planned Evolution
- Edit task text
- Filter by completed / uncompleted tasks
- Save tasks using localStorage
- Task counter
- Drag & drop reordering

## Step by Step Implementation (Pseudo-code)

### Add Task
1. Get input value
2. Validate input is not empty
3. Create a new <li> element
4. Add task text
5. Add "Delete" button
6. Add "Complete / Incomplete" toggle button
7. Append the <li> to the task list
8. Clear input field

### Delete Task
1. Identify clicked task
2. Remove it from the list

### Toggle Completion
1. Identify clicked task
2. Add / remove "completed" class
3. Update styling accordingly
